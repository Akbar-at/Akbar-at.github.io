Line Sensor PCB Design

November 2019 - December 2019

This PCB that I designed is a line sensor that is used to make autonomous robots. I designed this PCB with the Altium Software. I also printed the PCB and soldered all the parts together. This sensor uses the infrared led to figure out if there is a black line in front of the sensor. Because black color dont reflect the infrared light, the sensor will notice that using the voltage divider.The different values in voltage will be detected by the comparator IC LM358N.This sensor is a digital output sensor. The sensitivity of this sensor is fully controlled by the embedded potentiometer in the PCB itself.

Arduino Based Digital Scale

Time of Development : May 2021 - June 2021

This digital scale is a handmade Arduino digital scale project. The main processing unit is Arduino. This scale uses a load cell as the pressure transducer and lcd for the display. This scale is designed to be use for maximum load of 10 kilograms and have a gram resolution. This project also have a feature for price calculations which is commonly available for scale in supermarkets. It will output prices for the weight of the product. I worked on this project alone and it took me 1 month to build, design and troubleshoot this project.

Unmanned Surface Vehicle (MAKARA MH44XX)

Time of Development : July 2020 - December 2020

MAKARA MH4XX is an unmanned surface vehicle (USV). This project is a collaborative project under a research group called Autonomous Marine Vehicle Universitas Indonesia. I am the leader of the electrical engineering divison. On this project, I contributed on the design and fabrication of the electrical necessities of the vehicle such as wiring, troubleshooting, designing, parts tuning, and PCB design. This USV is a catamaran boat that uses two three-phase thrusters as the thrusting unit. It’s powered by two 12 Volt batteries for the processing units and two 16 Volt batteries for thrusting power. The main processing unit for this vehicle are Intel NUC where all the programming resides. The framework for the program uses ROS and it uses image detection so that it can work autonomously. PID controllers are used for the steering to make it balanced and accurate.

Autonomous Underwater Vehicle(MAKARA X)

Time of Development : January 2021 - Present

MAKARA X is an Autonomous Underwater Vehicle (AUV). This project is a collaborative project under a research group called Autonomous Marine Vehicles. I am the head leader of the electrical engineering division. On this project I contributed on the design and fabrication of the electrical side of the vehicle such as wiring, troubleshooting, designing, parts tuning, and PCB design. This AUV is a designed for the Singapore AUV Challenge. It’s equipped with 6 thrusters for maximum maneuvering underwater. Beside that, the main processing unit of this vehicle is the Nvidia Jetson Nano. We use this mini-pc because the vehicle performance will be greatly influenced by the processing power of the mini-pc. We use transfer learning algorithm on the vehicle so that it can detect objects autonomously and finish the missions that are given to us by the conductor of the competition.
